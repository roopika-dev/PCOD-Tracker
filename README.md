# 🌸 PCOD Tracker

PCOD Tracker is a health and wellness tracking web application designed to help users monitor PCOD/PCOS-related activities, symptoms, lifestyle habits, and health progress in a simple and user-friendly way.

---

## ✨ Features

- 🔐 User Authentication (Login & Register)
- 📊 Dashboard Overview
- ❤️ Daily Health Tracker
- 📅 Period & Cycle Tracking
- 📖 Health History Records
- 📱 Mobile Responsive UI
- 🌐 Progressive Web App (PWA) Support
- 🔥 Firebase Integration
- 🎨 Modern Clean UI Design
